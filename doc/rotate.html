Rotate

Purpose: Rotates the image, for straightening the horizon or a dominant vertical line. Four modes:

  • Rotate 90, 180, 270: Rotates the image clockwise in the number of degrees specified. When any of these is selected, the Rotate < 45 preview is hidden.
  • Rotate < 45:The slider rotates the preview image between -45 and +45 degrees in .1 degree increments. The rotation is applied when the mouse button is released. The autocrop checkbox enables automatic post-rotate cropping of the image to maximum usable image; the preview shows the boundaries on the uncropped rotated image.

Properties: See tool.rotate.* in Configuration Properties
