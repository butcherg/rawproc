Lens Vignetting

The lens vignetting tool offers the ability to manually enter vignetting correction parameters to apply the correction. This tool is made available in the menu when the tool.lensvignetting.enable property is set to 1.

Two algorithms are currently available, lensfun pa, and Adobe FixVignetteRadial. Parameters for both algoritims can be entered; only the paramters for the selected mode will be applied.

A paramter reset button is available for each mode. It will reset the paramters to those initally loaded in the tool.

Properties: See tool.lensvignetting.* in Configuration Properties
